**Tag** — muted outline capsule that classifies an event by its theme keywords. Lowercase, low-emphasis; render them in a wrapping row beneath an event body.

```jsx
<Tag>food</Tag>
<Tag>suspicion</Tag>
<Tag>ritual</Tag>
<Tag>apocalyptic</Tag>
```

Tags are descriptive, never interactive. For colored gain/cost/threat labels use `ConsequenceChip` instead.
